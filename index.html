Mixlist

AI Collaborative Playlist Generator

Overview

Mixlist is a system that generates collaborative playlists using listener preferences while avoiding genre conflicts.

Problem Statement

Spotify Blend playlists often introduce genres that some listeners do not enjoy. Mixlist analyzes listener patterns to create playlists that maintain variety without introducing unwanted genres.

System Architecture

Frontend: HTML / CSS
Backend: FastAPI
API Integration: Spotify
Data Processing: Python

Technology Stack
  • Python
  • FastAPI
  • Spotify API
  • REST Architecture
